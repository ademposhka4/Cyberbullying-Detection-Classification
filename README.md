/_config.yml: Jekyll site configuration (site title, theme, URL, and base URL for Enterprise Pages)

/index.md: Coordinates the website's landing page and navigation between tabs

/project-proposal.md: Project Proposal page with tab navigation; permalink `/project-proposal/`

/midterm.md: Midterm page  with tab navigation; permalink `/midterm/`

/README.md: Repository README (this file), describing directories and files
